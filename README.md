## Overview
- Universe: NIFTY 100
- Frequency: Monthly rebalance (trading month-ends)
- Stock selection: Benchmark-relative, volatility-adjusted momentum
- Portfolio construction: CAPM-implied expected returns with Max-Sharpe optimization
- Cash allocation: Explicit risk-free allocation when equity excess returns are unattractive

## Risk & Evaluation
- Performance attribution: CAGR, Sharpe, Sortino, Alpha, Beta
- Drawdown analysis
- Tail risk: Historical VaR / CVaR
- Monte Carlo simulation and stress testing
